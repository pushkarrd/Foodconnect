## üîß Setup Instructions (5 Minutes)

### Step 1: Get Firebase Credentials
1. Go to [Firebase Console](https://console.firebase.google.com/)
2. Click "Create Project"
3. Name it "FoodConnect"
4. Click "Create"
5. Wait for setup completion
6. In project settings ‚Üí copy all credentials

### Step 2: Enable Firebase Services
1. **Authentication**: Go to Authentication ‚Üí Sign-in method ‚Üí Enable "Email/Password"
2. **Firestore**: Click "Create Database" ‚Üí Start in test mode
3. **Storage**: Click "Create Bucket" (keep default settings)
4. **Realtime Database**: Click "Create Database"

### Step 2B: Create Firestore Collections
In your Firestore Database, create these 3 collections:

#### Collection 1: `users`
**Document ID**: User UID (auto-generated by Firebase Auth)
**Fields**:
```
uid: string (user ID)
name: string
email: string
phone: string
role: string (Donor or Receiver)
address: string
location: {
  latitude: number,
  longitude: number
}
ratings: array
averageRating: number
totalDonations: number
createdAt: timestamp
profileComplete: boolean
```

#### Collection 2: `donations`
**Document ID**: Auto-generated
**Fields**:
```
donorId: string (user ID of donor)
foodType: string (e.g., "Rice", "Bread", "Cooked Meal")
quantity: string (e.g., "5 kg")
description: string
imageURL: string (URL from Firebase Storage)
preparationTime: string
pickupTime: string
location: {
  latitude: number,
  longitude: number
}
address: string
status: string (Pending, Booked, Collected, Cancelled)
receiverId: string (null until booked)
createdAt: timestamp
expiresAt: timestamp (12 hours from creation)
bookedAt: timestamp (optional)
collectedAt: timestamp (optional)
cancelledAt: timestamp (optional)
```

#### Collection 3: `bookings`
**Document ID**: Auto-generated
**Fields**:
```
donationId: string (reference to donation)
receiverId: string (user ID of receiver)
status: string (Active, Completed, Cancelled)
bookedAt: timestamp
```

### Step 3: Configure Your App
1. Open `.env` file in the project root
2. Copy your Firebase credentials:
```
VITE_APP_FIREBASE_API_KEY=your_api_key_here
VITE_APP_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_APP_FIREBASE_PROJECT_ID=your_project_id
VITE_APP_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
VITE_APP_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_APP_FIREBASE_APP_ID=your_app_id
VITE_APP_GOOGLE_MAPS_API_KEY=optional_for_now
```

### Step 4: Run the Project
```bash
cd c:\Users\Pushkar\web projects\foodconnect
npm run dev
```

Visit: `http://localhost:3000`
